### 🏛️ 民法总论 — 第十二章 民事法律行为制度概述

#### 第一节 民事法律行为的概念和特征

##### 一、民事法律行为的概念

**【课本原文】** 《民法总则》第133条："民事法律行为是民事主体通过意思表示设立、变更、终止民事法律关系的行为。"

**【课本原文】** 《民法总则》从以下两个方面改变了《民法通则》的规定：
- 一是**取消了合法性的要求**，因为民事法律行为即便非法，也并非当然无效。在许多情况下，即使民事主体实施了欺诈等非法行为，如果受害人未提出撤销该行为，则该行为也应当有效。
- 二是**增加了意思表示的概念**，即强调民事主体要通过意思表示实施民事法律行为，这就揭示了民事法律行为的本质特征。

> 📌 **法律行为（Rechtsgeschaeft）** 的概念最早由德国学者古斯塔夫·胡果（Gustav Hugo，1764—1844）在1805年出版的《日耳曼普通法》一书中提出，并为《德国民法典》所采纳。

##### 二、民事法律行为的特征

**【课本原文】** 三个特征：

**（一）民事主体实施的以设立、变更、终止民事法律关系为目的的行为**
- 必须由民事主体实施（法院判决、仲裁裁决不是民事法律行为）
- 产生民事权利义务关系的设立、变更和终止
- **不限于合法行为**，非法的行为也可能发生法律效果（如欺诈可能产生合同被撤销等法律效果）

**（二）通过意思表示而实施的行为**
- 意思表示：向外部表明意欲发生一定私法上效果之意思的行为
- 表意人将其内心意思表示于外，为他人所知晓
- **这是法律行为与事实行为的根本区别**：事实行为不以意思表示为要件，其法律后果由法律直接规定

**（三）能够产生当事人预期的法律效果的行为**
- 法律行为并不是可以产生任何法律上的效果，而仅仅是**可以产生私法上的效果**
- 不仅产生私法上的效果，而且能够产生**当事人所预期的法律效果**
- 这也是法律行为与事实行为的重要区别

**【课本原文】** 本书总结：民事法律行为是民事主体实施的、以意思表示为要素、且能够产生当事人预期效果的行为。

##### 三、民事法律行为的成立

**【课本原文】** 《民法总则》第134条第1款："民事法律行为可以基于双方或者多方的意思表示一致成立，也可以基于单方的意思表示成立。"

| 行为类型 | 成立条件 |
|----------|----------|
| 单方法律行为 | 根据是否需要受领区分（需受领→到达时；无需受领→作出即生效） |
| 双方法律行为 | 双方意思表示一致（如合同） |
| 多方法律行为 | 多方当事人意思表示一致 |
| 决议行为 | 依法律或章程规定的议事方式和表决程序（《民法总则》第134条第2款） |

**【课本原文】** 民事法律行为一旦成立，即对当事人产生拘束力，任何一方当事人都不得单方面否定该民事法律行为的效力。成立是生效的前提。

##### 四、与相关概念的区别

| 概念 | 区别要点 |
|------|----------|
| **事实行为** | 法律行为以意思表示为要素；事实行为不以意思表示为要素，法律后果由法律直接规定 |
| **情谊行为** | 情谊行为无受法律拘束的意思，不产生法律效果（如搭顺风车约定） |

---

#### 第二节 民事法律行为的分类

**【课本原文】** 共八类：

| 序号 | 分类 | 核心区别 | 示例/要点 |
|------|------|----------|-----------|
| 一 | **单方 vs 双方 vs 共同** | 意思表示方向与数量 | 单方：遗嘱；双方：合同；共同：章程、合伙 |
| 二 | **一般 vs 决议** | 主体/生效/程序不同 | 决议实行多数决，对未参与者也有拘束力 |
| 三 | **有偿 vs 无偿** | 是否互为对价给付 | 无偿中注意义务标准较低 |
| 四 | **诺成 vs 实践** | 是否需交付标的物 | 诺成："一诺即成"；实践：还需交付（定金/保管） |
| 五 | **要式 vs 不要式** | 是否需特定形式 | 保证合同须书面形式 |
| 六 | **主行为 vs 从行为** | 是否独立存在 | 从行为有发生/效力/转让/消灭四方面从属性 |
| 七 | **独立 vs 辅助** | 是否有独立实质内容 | 辅助：法定代理人同意 |
| 八 | **有因 vs 无因** | 是否以原因为有效要件 | 无因：票据行为；有因：绝大多数法律行为 |

> ⚠️ **重要区分**：共同行为需要当事人意思表示一致，而决议行为可能实行多数决，并不当然需要一致。

#### 第三节 民事法律行为的形式

**【课本原文】** 《民法总则》第135条："民事法律行为可以采用书面形式、口头形式或者其他形式；法律、行政法规规定或者当事人约定采用特定形式的，应当采用特定形式。"

| 形式类型 | 要点 |
|----------|------|
| **书面形式** | 合同书、信件、电报、电传、传真、微信、QQ等（数据电文属于书面形式——《合同法》第11条） |
| **口头形式** | 即时结清交易最普遍；简便易行但举证困难 |
| **其他形式** | 主要指**默示形式**——通过一定行为作出意思表示（如续租后继续交租金、自动售货机投币） |

**【课本原文】** 如果法律、行政法规没有对违反民事法律行为形式的法律后果作出规定，则从鼓励交易、保护当事人的私法自治出发，不应当轻易否定当事人民事法律行为的效力。

#### 第八章第六节 刑法上的认识错误

**【课本原文】** 刑法上的认识错误，是指行为人对于自己行为的法律性质或事实情况的认识发生错误。理论上一般分为两大类，即**法律认识错误**和**事实认识错误**。

##### 一、法律认识错误（违法性错误）

| 类型 | 【课本原文】含义 | 处理 |
|------|----------------|------|
| **(一) 假想的犯罪** | 行为在法律上不构成犯罪，行为人误认为构成犯罪 | 根据罪刑法定原则，只能依据法律认定，不以犯罪论处 |
| **(二) 假想的无罪** | 行为在法律上构成犯罪，行为人误认为不构成犯罪 | 原则上应依法追究，不懂法不是免责理由 |
| **(三) 对罪名及刑罚的认识错误** | 认识到行为构成犯罪，但罪名或刑罚认识错误 | 不影响定罪也不影响量刑 |

**【课本原文】** 假想无罪有三种例外，此时不宜认定为犯罪：
1. 某种原来并非法律所禁止的行为，后来以特别法规定为犯罪，在法律刚实施之际，行为人确实不知道新法已禁止此行为
2. 对法的状况产生疑问，因信赖主管机关或者司法机关的见解而产生违法性错误的
3. 当上位法和下位法发生矛盾时，行为人因信赖下位法而违背上位法的

##### 二、事实认识错误

**【课本原文】** 事实认识错误，是指行为人主观上对决定其行为性质及刑事责任的有关事实情况存在不正确的理解。

**【课本原文】** 学说：
- **具体符合说**：只有当行为人所认识的事实与实际发生的事实相一致时，才成立故意的既遂犯
- **法定符合说**：行为人所认识的事实与实际发生的事实，只要在犯罪构成范围内是一致的，就成立故意的既遂犯
- **通说和司法实践均采法定符合说** ⭐

| 类型 | 【课本原文】含义 | 处理要点 |
|------|----------------|----------|
| **(一) 客体错误** | 意图侵犯一种客体，实际上侵犯了另一种客体 | 按行为人意图侵犯的客体定罪 |
| **(二) 对象错误** | 对行为所指向的人或物的具体性质或种类的认识错误 | 分为三种情况（见下表） |
| **(三) 行为性质错误** | 对自己行为的实际性质发生错误认识（假想防卫/假想避险） | 一般排除故意，要么不构成犯罪，要么是过失犯罪 |
| **(四) 工具错误** | 对使用的工具产生错误认识（如错把白糖当砒霜） | 以犯罪未遂追究 |
| **(五) 因果关系错误** | 对因果关系有无、发展方向或具体进程发生错误认识 | 四种情况具体分析 |
| **(六) 打击错误** | 因行为本身偏差导致欲攻击对象与实际受害对象不一致 | 采法定符合说：同一构成内→既遂；不同构成→故意未遂+过失竞合 |

> **对象错误的三种情况：**
> 1. 意欲指向的对象不存在→犯罪未遂
> 2. 同一性质对象错误（同一犯罪构成内）→不影响定性（采法定符合说）
> 3. 不同性质对象错误（不同犯罪构成）→故意未遂与过失竞合

#### 第九章 正当行为

##### 第一节 正当行为概述

**【课本原文】** 正当行为，在我国刑法理论中又称为排除犯罪性行为，是指客观上造成一定的损害结果，形式上符合刑法对某种犯罪规定的客观构成要件，但**实质上既不具有社会危害性，也不具有刑事违法性**的行为。

| 特征 | 内容 |
|------|------|
| 特征一 | 形式上符合某种犯罪的客观构成要件 |
| 特征二 | 实质上并不符合某种犯罪的构成特征（不具备犯罪客体、犯罪主观要件等） |

**【课本原文】** 刑法明文规定：正当防卫和紧急避险两种。本书认为还包括：自救行为、正当业务行为（医疗/竞技）、法令行为、基于权利人承诺或自愿的损害行为。

---

##### 第二节 正当防卫 ⭐⭐

**【课本原文】** 《刑法》第20条第1、2款——正当防卫是指为了使国家、公共利益、本人或者他人的人身、财产和其他权利免受正在进行的不法侵害，而对不法侵害者实施的制止其不法侵害且未明显超过必要限度造成重大损害的行为。

**三大特征：**
1. 一般通过对不法侵害人的反击行为实施（具有杀人、伤害等行为表现）
2. 是针对不法侵害行为实施的正当、合法行为（受国家法律保护、支持和鼓励的私力救济）
3. 目的是使合法权益免受正在进行的不法侵害

###### 正当防卫的五个条件（缺一不可）

```
         起因条件         时间条件          对象条件          目的条件          限度条件
        "存在不法侵害"    "正在进行"      "针对侵害人本人"    "保护合法权益"    "未明显超过必要限度"
              │               │                │                │               │
     ┌───────┴───────┐  ┌─────┴─────┐   ┌──────┴──────┐  ┌─────┴─────┐  ┌──────┴──────┐
     │•违法+犯罪行为  │  │•已经开始    │   │•不法侵害人  │  │•防卫挑拨❌ │  │•不能明显超过│
     │•有暴力/紧迫性  │  │•尚未结束    │   │•及其共犯    │  │•互殴行为❌ │  │•综合判断    │
     │•客观实际存在   │  │•事前防卫❌  │   │•不含无关亲友│  │•偶然防卫❌ │  │•不要求相等   │
     │•对合法行为不行 │  │•事后防卫❌  │   │             │  │           │  │             │
     │•动物自发→避险  │  │             │   │             │  │           │  │             │
     └───────────────┘  └─────────────┘   └──────────────┘  └─────────────┘  └──────────────┘
```

###### 特殊防卫（无限防卫/无过当防卫）

**【课本原文】** 《刑法》第20条第3款："对正在进行行凶、杀人、抢劫、强奸、绑架以及其他严重危及人身安全的暴力犯罪，采取防卫行为，造成不法侵害人伤亡的，不属于防卫过当，不负刑事责任。"

- ⚠️ "严重危及人身安全的暴力犯罪" 修饰**全部六种**行为，非暴力手段（投毒杀人、麻醉抢劫等）**不能**实行特殊防卫
- "行凶"的认定：使用致命性凶器 / 虽未使用凶器但根据人数/打击部位/力度等已严重危及人身安全

###### 防卫过当

| 项目 | 内容 |
|------|------|
| 概念 | 防卫明显超过必要限度造成重大损害 |
| 罪过形式 | 只能是**间接故意或过失**，不能是直接故意 |
| 定罪 | 不是独立罪名，依主观罪过和客观行为定（过失致人死亡/重伤罪、故意杀人/伤害罪） |
| 量刑 | **应当减轻或者免除处罚** |

##### 第三节 紧急避险 ⭐

**【课本原文】** 《刑法》第21条第1款——紧急避险是指为了使国家、公共利益、本人或者他人的人身、财产和其他权利免受正在发生的危险，不得已而采取的损害另一较小合法权益的行为。

###### 紧急避险的六个条件（缺一不可）

| 条件 | 核心内容 | ⚠️ 关键点 |
|------|----------|-----------|
| **起因条件** | 合法权益遭受损害危险 | 危险来源：人的不法侵害、自然灾害、动物侵袭、人的生理/疾患 |
| **时间条件** | 危险正在发生 | 事前避险/事后避险 = 避险不适时 |
| **对象条件** | 第三者合法权益 | **区别于正当防卫**（正当防卫针对侵害人本人） |
| **目的条件** | 正当的避险意图 | 认识到只能通过损害较小权益排除危险 |
| **限度条件** | 损害必须**小于**所避免的损害 | 不能大于或等于；人身权利>财产权利 |
| **限制条件** | 不得已（最后手段） | 职务上/业务上负有特定责任的人**不能**为避免本人危险避险 |

###### 避险过当：损害大于或等于所保护的权益 → 应当减轻或者免除处罚

---

###### 紧急避险 vs 正当防卫对比（核心考点！）

| 维度 | 正当防卫 | 紧急避险 |
|------|----------|----------|
| 起因条件 | 仅限于**人的不法侵害** | 多种来源（人/自然/动物/生理） |
| 对象条件 | 针对**不法侵害人本人** ⭐ | 针对**第三者合法权益** ⭐ |
| 限度条件 | 未明显超过必要限度 | 损害**必须小于**所避免的损害 |
| 限制条件 | 无特殊限制 | **不得已**（最后手段） |
| 主体要求 | 无特殊限制 | 职务/业务上负有特定责任的人受限制 |

##### 第四节 其他正当行为

| 类型 | 【课本原文】要点 |
|------|-----------------|
| **自救行为** | 无法等待公权力救济时自力恢复权利；要件：不法侵害已造成侵害+来不及通过法律程序+具有自救目的+行为相当性 |
| **正当业务行为** | 依职业要求实施（医疗、竞技）；要件：业务合法+有益社会的目的+未超必要限度 |
| **法令行为** | 依法律/法规/上级命令实施（逮捕、执行死刑）；分类：依照法律实施的行为+执行命令的行为 |
| **基于权利人承诺的损害** | 经权利人同意后损害其利益；要件：有权处理+自愿+合法+有益社会+在授权范围内。⚠️ "安乐死"在我国仍构成犯罪 |

### 🧠 法律逻辑学 — 第八章 演绎推理

> ⚠️ 逻辑课本为扫描版PDF，以下内容基于已核验的思维导图结构整理。

#### 第一节 三段论

##### 三段论的规则 ⭐⭐

| 序号 | 规则 | 记忆提示 |
|------|------|----------|
| 1 | 中项必须至少在一个前提中**周延** | 中项不周延→无法建立大小项联系 |
| 2 | 前提中不周延的词项在结论中不得周延 | 结论不能超出前提范围 |
| 3 | 两个否定前提不能得出结论 | 否否无果 |
| 4 | 前提有一否定，结论必否定 | 一否则否 |
| 5 | 两个特称前提不能得出结论 | 特特无果 |
| 6 | 前提有一特称，结论必特称 | 一特则特 |

##### 三段论的格

```
第一格  M—P      第二格  P—M      第三格  M—P      第四格  P—M
        S—M              S—M              M—S              M—S
        ∴S—P             ∴S—P             ∴S—P             ∴S—P

法律适用中最常用第一格（大前提+小前提→结论）
```

##### 三段论的省略式
- 省略大前提（法律依据默认已知）
- 省略小前提（案件事实不言自明）
- 省略结论（不言而喻）

#### 第二节 复合命题推理

##### 核心公式表 ⭐⭐

| 推理类型 | 有效式 | 无效式 |
|----------|--------|--------|
| **充分条件假言** `p→q` | **肯定前件式**：p→q, p ∴ q | 否定前件式 |
| | **否定后件式**：p→q, ¬q ∴ ¬p | 肯定后件式 |
| **必要条件假言** `p←q` | **否定前件式**：p←q, ¬p ∴ ¬q | 肯定前件式 |
| | **肯定后件式**：p←q, q ∴ p | 否定后件式 |
| **相容选言** `p∨q` | **否定肯定式**：p∨q, ¬p ∴ q | **肯定否定式** ❌ |
| **不相容选言** | 否定肯定式 ✓ / 肯定否定式 ✓ | — |
| **联言推理** | 分解式：(p∧q)→p / 组合式：p,q→(p∧q) | — |

> 💡 **关键记忆点**：相容选言的肯定否定式是**常见谬误**——"p或q，已知p，不能推出非q"（因为可能两个都成立！）

---

#### 第三节 刑事侦查工作中正确运用演绎推理的方法
- 从现场痕迹到嫌疑人特征的推理
- 从证人证言到案件事实的推理
- 排除法在侦查中的应用

#### 第四节 法律推理的特征及运用
- **制定法三段论**：法律规范（大前提）+ 案件事实（小前提）→ 判决结论
- **类比法律推理**：类似案件类似处理
- **常见谬误**：循环论证、偷换概念、以偏概全

## 🧠 三、记忆诀窍

### 民法

| 知识点 | 口诀/技巧 |
|--------|-----------|
| 三特征 | **"目—意—效"**：目的性（设变终）+ 意思表示 + 预期效果 |
| 八分类 | **"单双共决，有无偿诺成要主独因"** → 单双方共同/决议/有偿无偿/诺成实践/要式不要式/主从/独立辅助/有因无因 |
| 单双方共同区别 | 单方一人说了算，双方交换意思看，多方同向共向前 |
| 实践合同记忆 | **"定金、保管、自然人借款"** — 三样需交付才成立 |
| 从行为四从属性 | **"发生效转灭"** — 发生上、效力上、转让上、消灭上 |

### 刑法

| 知识点 | 口诀/技巧 |
|--------|-----------|
| 法律认识错误三类型 | **"假有罪、假无罪、罪名刑期搞错"**：假想犯罪→假想无罪→罪名刑罚错 |
| 事实认识错误六类型 | **"客体对象性，工具因果打"**：客体错误→对象错误→行为性质错误→工具错误→因果关系错误→打击错误 |
| 正当防卫五条件 | **"起因时对象限度"** → 起时对限度（qi shi dui xian du）|
| 紧急避险六条件 | 正当防卫五条件 + **"限制条件（不得已）+ 主体限制"** |
| 防卫 vs 避险核心区别 | "防卫打**人**，避险打**第三人**"；"防卫够用就行，避险**必须更小**" |
| 特殊防卫记忆 | **"行杀抢强绑"** — 都必须是**暴力**手段！投毒/麻醉≠特殊防卫 |
| 过当罪过形式 | 防卫过当/避险过当—**不能是直接故意**（正当目的与直接故意矛盾） |

### 逻辑

| 知识点 | 口诀/技巧 |
|--------|-----------|
| 三段论六规则 | **"中周延、大不小、否否否、一否则否、特特特、一特则特"** |
| 充分条件有效式 | **"肯前否后"** — 肯定前件 + 否定后件有效 |
| 必要条件有效式 | **"否前肯后"** — 与充分条件**正好相反**！ |
| 相容选言唯一无效 | **"不能肯否"** — 相容选言的肯定否定式是无效的 |
| 两者对称记 | 充分=肯前否后；必要=否前肯后（**颠倒过来**） |

## 💡 六、今日三科串联提示

> 🔗 **"表意—认识—推理"三元串联**
>
> 今天三科有一条暗线串起来：**意思表示（民法）→ 认识错误（刑法）→ 三段论推理（逻辑）**。
>
> - 民法：民事法律行为的核心是**意思表示**——表意人将内心意思表达于外
> - 刑法：认识错误的核心是**主观认识**与客观事实不符——行为人"意思"出了偏差
> - 逻辑：三段论的核心是从**已知命题**推出**未知结论**——认知的扩展
>
> 三条线交汇在"人的主观认知如何与客观世界对接"这一法哲学命题上。
>
> 试试在复习时思考：**民法的"意思表示真实"是法律行为生效要件，刑法的"认识错误"是影响罪过的因素——本质上都在处理"主观意思与客观事实的关系"问题。**
>
> 逻辑学的三段论则给了我们一套**形式化的工具**来判断这种"主客观对接"是否有效。
